~ = tilde
' = (simple) quote
" = double quote
[ ] = bracket
{ } = accolades
& = esperluette
( ) = parenthèse
| = pipe
# = dièse / sharp
`` = magic quotes
; = semicolon
++ / +1 = incrémentation
-- / -1 = décrémentation

Les commandements des développeurs :
- Une variable tu ne créeras point, en revanche une variable tu déclareras
- Tu ne donneras point de valeur à une variable, tu l'initialiseras
- Une variable dans les parenthèses d'une fonction n'existe pas, mais un paramètre existera
- De beaucoup de café et d'efferalgans besoin tu auras
